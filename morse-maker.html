I built the Morse Maker because I wanted a standalone, affordable, professional-quality Morse code trainer for myself. I'm proud of what I built, I use it everyday, and now I'm excited to offer it to the world

Features

  • SEND PRACTICE - Send morse code (using onboard capactive touch keyer, external straight key, or external paddle) and watch it decoded live.
  • LISTEN PRACTICE - Over 40 builtin lessons using the Koch method. Start from just 2 characters and add new characters after mastering each lesson, eventually progressing to the whole alphabet. Listen to over 1000 english words or groups of 1/3/5 characters.
  • AUDIO-ONLY LISTEN PRACTICE - Contains recordings of all 1000 built-in english words, characters, and special characters. The correct answer can be voiced aloud after playing the Morse code. Focus solely on the audio without needing to look at a screen.
  • LOCAL RADIO - Two way 2.4GHz communication allows sending and recieving Morse code up to 100m (~330ft). Practice with nearby friends and family. Sent and recieved morse can optionally be decoded on the screen live.
  • CONFIGURABLE - Adjust WPM (5-40 WPM), Farnsworth effective WPM (5-40 WPM), keyer Type (onboard straight key, external straight key, or external paddle), tone frequency (400-1000 Hz), and RF channel (for local radio).

Hardware

  • ENCLOSURE - 3D printed PET-G enclosure.
  • DISPLAY - Vibrant 2.42 inch OLED display. Very large and easy to read text.
  • CAPACITIVE KEYER - Built-in capactive touch sensor straight key directly on the enclosure. No external keyer required.
  • BATTERY - 800mAh Lithium Ion rechargable battery provides 10 hours of practice on a single charge. Recharge with supplied USB-C charging cable.
  • SPEAKER - Listen using the included speaker (not a buzzer!) or plug in headphones using the 3.5mm jack.
